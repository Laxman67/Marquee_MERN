





What is CSS Grid and Why Do We Use It – Introduction to CSS Grid, two-dimensional layouts, advantages, and use cases.

*Grid Container vs Grid Items – Difference between the parent grid container and its child elements.

*Grid Template Columns & Rows – Defining the number and size of columns and rows.

*Grid Gap (gap, row-gap, column-gap) – Adding spacing between rows and columns.

*Fraction (fr) Unit – Creating flexible and responsive column and row sizes.
*Grid Lines & Item Placement – Positioning items using grid lines and spans.
*Repeat Function (repeat()) – Simplifying repetitive column and row definitions.

*Justify Items – Aligning grid items horizontally within their cells.
*Align Items – Aligning grid items vertically within their cells.
*Justify Content – Aligning the entire grid horizontally inside the container.
*Align Content – Aligning the entire grid vertically inside the container.

Minmax Function (minmax()) – Setting minimum and maximum sizes for responsive layouts.
